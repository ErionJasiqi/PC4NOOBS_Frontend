The idea behind our “PC4NOOBS” project is to develop a web-based application that allows users to configure and build their own PC. The application is designed specifically to help beginners select suitable hardware components and avoid common problems. The system automatically pciks the most suitable components based off of the questionnaire and calculates the total cost of the build.
